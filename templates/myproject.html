Dynamic Traffic Lights

What?

Dynamic Traffic Lights is an approach to automate the traffic signals through object detection. Dynamic Traffic Lights is a computer software program with the ability to detect the object (vehicles). It is a project based on Object detection technique. It requires training before it can be used, which requires you to install the system in the cameras. Initially, when the traffic signal turns to red, and 5 seconds prior when the signal turns to green, the object detection techniques starts counting number of vehicles and generate dynamic traffic time according to the number of vehicles and traffic density. Hence due to this system, traffic efficiency will improve and will be beneficial to everyone.

Why?

Our main objective is to consider this issue as our project definition and to reduce delay and help people to commute to their respective destination with safety and without burden of traffic congestion. It is a computer software program with the ability to detect the object (vehicles). It is a project based on Object detection technique. It requires training before it can be used, which requires you to install the system in the cameras.With the help of this project, it will help to conduct smooth simulation of the vehicles.

How?

Traffic congestion is vital problem in every country, state, and city. Traffic congestion means vehicles travel slowly as there is too much traffic on every road. There are various other options to reduce traffic congestion, like cars can be banned in few districts or allowed during specific time, or people commute through public transport, but all these things are very difficult to adopt in big cities. So, the only solution is to reduce the traffic congestion. Congestion can also lead to reduction in road accidents as vehicles will commute very slowly. On the other hand, due to this congestion people won’t be able to travel to their respective destination. We cannot stop road accidents but we surely can try to reduce it with smooth simulation of vehicles during cross roads and hence, our project on Dynamic Traffic light will help us to reduce both issues.

Differences

  • Static Traffic lights
  • 1. Static traffic lights has predefined timer for a particular light.
  • 2. Static traffic lights doesn't store any data.
  • 3. Static traffic lights doesn't count the number of vehicles
  • Dynamic Traffic lights
  • 1. Dynamic traffic lights doesn't have any predefined timer. It calculates according to the count of vehciles present at given point of time.
  • 2. Dynamic traffic lights store information of count of vehicles which can also be used in traffic predictions, at a particular time.
  • 3. Dynamic traffic lights counts the number of vehciles and gives timer based on count of vehicles.
